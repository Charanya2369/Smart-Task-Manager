# Smart-Task-Manager
AI-powered Smart Task Manager built with Spring Boot, Maven, Java, REST APIs, and modern web technologies to improve productivity through intelligent task management, analytics, and achievement tracking.

## 📌 Overview

TaskMentor is a full-stack productivity management application that helps users efficiently organize daily tasks while providing intelligent productivity insights.

The system combines task scheduling, progress tracking, analytics, achievement badges, and an AI mentor to encourage users to stay productive and maintain consistency.

---

## ✨ Features

- ✅ Secure User Authentication
- ✅ Create, Update and Delete Tasks
- ✅ Smart Task Categorization
- ✅ Priority Management
- ✅ Due Date & Time Scheduling
- ✅ Productivity Analytics Dashboard
- ✅ Completion Statistics
- ✅ Daily Productivity Grade Report
- ✅ Achievement & Streak System
- ✅ AI Mentor Motivation Panel
- ✅ Search & Filter Tasks
- ✅ Responsive Dashboard
